# SmartHome — בית חכם

## זהות הפרויקט
- שם: SmartHome — בית חכם
- מטרה: אפליקציית ניהול משק בית משפחתי מסונכרנת בזמן אמת בין כל בני המשפחה
- פלטפורמה: Web (React + Vite) — בהמשך Android
- שפת ממשק: עברית — RTL מלא
- יעד: פרסום ב־Google Play Store (שלב שני)

## סטאק טכנולוגי
- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: MongoDB Atlas
- Images: Cloudinary (cloud: dhrgdc4ne, preset: smarthome_uploads)
- Auth: Google OAuth 2.0
- Deploy: Render

## מודולים (לפי סדר פיתוח)
1. אימות משתמשים + כניסה עם Google
2. דשבורד ראשי למנהל
3. משימות בית — ביצוע + אישור בזמן אמת
4. רשימת קניות — בזמן אמת
5. לוח שנה + אירועים
6. מעקב מתנות — מחובר ליומן אוטומטית
7. ניהול תקציב
8. מלאי מזון
9. תורים לרופא + תיק רפואי
10. ניהול רכב
11. גמיפיקציה — נקודות ותגמולים לכל בני המשפחה
12. מערכת התראות

## מערכת משתמשים
- מנהל: אבא ו/או אמא — עד 2 מנהלים — הרשאות מלאות
- משתמש רגיל: ילדים ובני משפחה — הרשאות מוגבלות
- כניסה: חשבון Google אישי לכל משתמש
- מנהל מאשר כל משתמש חדש

## עדכון בזמן אמת — חוק ברזל
כל פעולה חייבת:
- להתעדכן ב־MongoDB מיידית
- לשמור: שם המבצע + תאריך + שעה
- להיות גלויה לכל בני הבית מיידית

## מודול מתנות — כלל מיוחד
- חיבור אוטומטי בין אירוע ביומן לרשימת המתנות
- מציג: מה ניתן/התקבל + תאריך האירוע האחרון
- חל איסור מוחלט להציג המלצות סכום
- התראה אוטומטית 7 ימים לפני כל אירוע

## גמיפיקציה — כללים
- כל משימה שבוצעה = נקודות אוטומטיות לפי עדיפות:
  - דחוף = 30 נקודות
  - רגיל = 20 נקודות
  - יכול לחכות = 10 נקודות
- לוח מובילים שבועי לכל בני המשפחה
- איפוס אוטומטי כל שבוע
- תגים: "מנקה השבוע", "אלוף הקניות", "מלך המשימות"

## כללי קוד
- TypeScript strict — אין any
- Functional components בלבד
- אין לוגיקה עסקית בקומפוננטות — רק ב־hooks ו־services
- כל קריאה ל־MongoDB דרך services/ בלבד
- הערות בקוד בעברית
- RTL מלא — אין hardcode של left/right
- כל טקסט למשתמש בקובץ utils/strings.ts בלבד

## התראות
- אירוע ביומן: 7 ימים לפני + יום לפני
- מתנה: 7 ימים לפני האירוע
- תקציב: כשמגיעים ל־80% מהתקרה
- מלאי: כשכמות יורדת מתחת לסף
- תרופות: כל יום בשעה שנקבעה
- טסט/ביטוח רכב: חודש לפני + שבוע לפני
